Introduction

Coding is how we tell computers what to do — step by step. Teaching kids to code helps them learn creativity, logic, and problem-solving. For example, typing alert("Hello!") makes a pop-up appear.

In this guide, we’ll explore the basics of programming in a simple, hands-on way that’s perfect for beginners.

Getting Started

You only need a web browser and a text editor to begin coding! Try using replit.com or codepen.io to run code online.

<h1>Hello, world!</h1>

This is your first line of HTML — the building block of webpages.

Variables and Data

Variables store information we can use later, like let score = 100; or const name = "Liora";.

let name = "Liora";
let age = 10;
console.log("Hi " + name + "! You are " + age + " years old.");

Try changing the values to see how the output changes!

Loops and Logic

Loops let us repeat actions automatically. For example, print numbers 1–5:

for (let i = 1; i <= 5; i++) {
  console.log(i);
}

Kids can think of this like giving instructions to a robot — repeat until done!

Functions

Functions are mini-programs we can reuse. For example:

function greet(name) {
  console.log("Hello, " + name + "!");
}
greet("Coder");

Functions help organize code into logical steps, like making recipes.

Projects and Practice

Now that you know the basics, try these fun projects:

  • Make a “Guess the Number” game
  • Create a digital birthday card with HTML and CSS
  • Design an interactive quiz using JavaScript
  • Animate your name using CSS transitions
  • Build a mini website about your favorite hobby

Practice, experiment, and most importantly — have fun coding!
